# BakaPotatoPatchMod

Minecraft 客户端补丁 Mod,支持 Fabric,Forge,NeoForge.

功能:

* 与 BakaPotatoPatcher Paper 插件进行客户端识别.
* 按服务器范围启用补丁:所有服务器或笨蛋土豆服务器.
* 修正异常 Waypoint 数据包导致的网络协议错误.
* 修正过多字节数据包导致的断连.
* 可自动同意服务器资源包.
* 提供客户端配置界面和游戏内 HUD 调试显示.
* 按服务器命令请求发送 Mod 列表和硬件 ID 哈希.
* 启动时和手动检查 Mod 更新.



Tips: 部分功能在没有服务器插件时不会正常工作, 开源Mod源代码仅用于表明该Mod不会对玩家的电脑做出非道德的操作.
